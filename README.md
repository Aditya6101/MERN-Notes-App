# MERN Notes App

A simple web application for taking short notes.


## Tech Stack

**Client:** React + Typescript, Redux Toolkit, TailwindCSS

**Server:** Node, Express, JWT Token

**Database:** MongoDB with Mongoose


## Features

- Authentication using JWT tokens
- Minimal UI
- Mobile Friendly
